# Notification Icons

This directory should contain the icons used for push notifications.

## Required Icons

### icon-192x192.png
- **Size**: 192x192 pixels
- **Format**: PNG
- **Purpose**: Main notification icon
- **Usage**: Shown in push notifications across all platforms

### badge-72x72.png
- **Size**: 72x72 pixels
- **Format**: PNG
- **Purpose**: Badge icon (monochrome)
- **Usage**: Shown in Android notification badges

## How to Create

You can create these icons using:
1. Any graphic design tool (Figma, Photoshop, Canva)
2. Online icon generators
3. Your existing logo/branding

### Tips:
- Use your Fire Safety logo or "FS" initials
- Keep it simple and recognizable at small sizes
- Use high contrast colors
- Make the badge icon monochrome (white on transparent)

## Temporary Solution

For development/testing, you can use placeholder icons:
1. Create a simple square with your brand color
2. Add "FS" text in white
3. Export at 192x192 and 72x72 sizes

The notification system will work without these, but won't show custom icons.
